# MANUFACTURING TRACKER
A web based tracking system for assembly of various 'child' parts with the 'parent' machine to record which parts were assembled for verification before delivery and to check presence of all child parts to check warranty on that product.

## TECH STACK
- Backend: Node.js + Express
- Database: PostgreSQL + Prisma
- Frontend: React (Phase 5)

## PROJECT STATUS
Phase 0 - Environment Setup

## ASSUMPTIONS
- USB barcode scanner (keyboard input)
- 15-character structured part barcodes
- Online first, offline sync added in Phase 8
